# congress-voting-predictor
A set of Machine Learning that predict the party to which a voter belongs to, based on data gathered from 433 US House of Representatives Congressmen's votes on a series of different topics in a 1984 Congress debate.

Dataset sourced from UCI Machine Learning Repository (http://archive.ics.uci.edu/ml/datasets/Congressional+Voting+Records)

Algorithms implemented:
1. Kernel SVM - F1 score of 0.94
2. Logistic Regression - F1 score of 0.95
3. Random Forest - F1 score of 0.92
4. Support Vector Machine - F1 score of 0.93
